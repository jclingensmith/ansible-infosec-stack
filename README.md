# ansible-infosec-stack
Ansible playbook and roles to provision a distro for infosec

![](./doc/ansible-logo.png)


### Requirements

```
Ansible 2.9.+
```

### Tools provisioned

* DNSutils
* BinWalk
* Hydra
* John the Ripper
* Masscan
* Metasploit-framework
* Netcat
* Nmap
* SQLmap
* TCPdump
* Whois
* Docker
